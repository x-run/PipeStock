## 設計原則

- **トランザクション駆動**: 在庫数量はトランザクション (Tx) の合計から算出する。在庫を直接編集してはならない。
- **単一倉庫**: MVP ではロケーション管理は行わない。
- **数量最優先**: 数量 > 単価 > 重量 の優先順位で設計する。
